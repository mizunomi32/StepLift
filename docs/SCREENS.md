# StepLift - 画面設計

## 画面一覧

| 画面 | パス | 説明 |
|------|------|------|
| ダッシュボード | `/(tabs)/` | ホーム画面、今日のサマリー |
| ワークアウト | `/(tabs)/workout` | ワークアウト記録メイン |
| 歩数 | `/(tabs)/steps` | 歩数トラッキング |
| 履歴 | `/(tabs)/history` | 過去の記録一覧 |
| 新規ワークアウト | `/workout/new` | ワークアウト開始 |
| ワークアウト詳細 | `/workout/[id]` | 進行中/過去のワークアウト |
| 種目選択 | `/workout/exercise-picker` | 種目選択モーダル |
| 設定 | `/settings` | アプリ設定 |

## 画面詳細

## ナビゲーション構造

```
app/
├── (tabs)/
│   ├── _layout.tsx      # タブナビゲーション
│   ├── index.tsx        # ダッシュボード
│   ├── workout.tsx      # ワークアウト
│   ├── steps.tsx        # 歩数
│   └── history.tsx      # 履歴
├── workout/
│   ├── new.tsx          # 新規ワークアウト
│   ├── [id].tsx         # ワークアウト詳細
│   └── exercise-picker.tsx  # 種目選択
├── settings/
│   └── index.tsx        # 設定
├── _layout.tsx          # ルートレイアウト
└── modal.tsx            # 汎用モーダル
```

## インタラクション

### ハプティクス
- ボタンタップ: Light impact
- セット完了: Medium impact
- 目標達成: Success notification
- 削除: Warning

### アニメーション
- 画面遷移: Shared element transition (可能な場合)
- プログレスリング: Spring animation
- リスト項目: Fade in / slide up
- ボタン: Scale on press
