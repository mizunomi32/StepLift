# Issue #008: ダッシュボード画面

## 概要
ホーム画面となるダッシュボードを実装する。今日のサマリー、週間統計、ストリーク表示を含む。

## タスク

### 1. ダッシュボード画面
`app/(tabs)/index.tsx`:

- ヘッダー（アプリ名、設定アイコン）
- 今日の歩数カード
- 今日のワークアウトカード
- 週間サマリーカード
- クイックアクションボタン

### 2. 今日の歩数カード
`components/dashboard/TodayStepsCard.tsx`:

```tsx
interface TodayStepsCardProps {
  steps: number;
  goal: number;
  streakDays: number;
}
```

表示:
- ミニプログレスリング
- 歩数 / 目標
- 達成率（%）
- ストリーク表示（🔥 X日連続）

### 3. 今日のワークアウトカード
`components/dashboard/TodayWorkoutCard.tsx`:

```tsx
interface TodayWorkoutCardProps {
  workout: Workout | null;
  onStartWorkout: () => void;
}
```

**ワークアウトがない場合:**
- 「今日のワークアウト」タイトル
- 「まだ記録がありません」メッセージ
- 「ワークアウトを開始」ボタン

**ワークアウトがある場合:**
- ワークアウト時間
- 種目数、セット数
- 主な種目名

### 4. 週間サマリーカード
`components/dashboard/WeeklySummaryCard.tsx`:

```tsx
interface WeeklySummaryCardProps {
  workoutCount: number;
  averageSteps: number;
  streakDays: number;
}
```

表示:
- 📊 ワークアウト: X回
- 👟 平均歩数: X,XXX歩
- 📅 ストリーク: X日

### 5. ストリーク計算ロジック
`lib/utils/streak.ts`:

```typescript
// 連続トレーニング日数を計算
export function calculateWorkoutStreak(workouts: Workout[]): number;

// 連続歩数目標達成日数を計算
export function calculateStepStreak(records: StepRecord[], goal: number): number;
```

### 6. 設定画面への導線
ヘッダーの設定アイコンから `/settings` へ遷移。
