## AIエージェントへの依頼方法

各issueファイルには以下の情報が含まれています：

1. **概要**: タスクの目的
2. **タスク**: 具体的な実装内容
3. **受け入れ条件**: 完了の判断基準
4. **参照ドキュメント**: 関連する設計ドキュメント
5. **依存関係**: 先に完了すべきissue
6. **見積もり**: 作業量の目安

### 依頼例

```
Issue #005 のワークアウト記録画面を実装してください。

参照ドキュメント:
- docs/SCREENS.md
- docs/DATA_MODEL.md

受け入れ条件を満たすように実装をお願いします。
```

## 進捗管理

issueの進捗は以下のステータスで管理できます：

- ⬜ Todo: 未着手
- 🔄 In Progress: 作業中
- ✅ Done: 完了
- ⏸️ Blocked: ブロック中
