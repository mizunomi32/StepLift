# Issue #010: 設定画面

## 概要
アプリの設定画面を実装する。目標設定、表示設定、データ管理、ヘルス連携を含む。

## タスク

### 1. 設定画面
`app/settings/index.tsx`:

- 目標セクション
- 表示セクション
- データセクション
- ヘルス連携セクション
- アプリ情報セクション

### 2. 設定ストア
`lib/stores/settings-store.ts`:

```typescript
interface SettingsState {
  // 目標
  dailyStepGoal: number;

  // 表示
  theme: 'light' | 'dark' | 'system';
  weightUnit: 'kg' | 'lb';

  // アクション
  setDailyStepGoal: (steps: number) => Promise<void>;
  setTheme: (theme: 'light' | 'dark' | 'system') => void;
  setWeightUnit: (unit: 'kg' | 'lb') => void;
  loadSettings: () => Promise<void>;
}
```

### 3. 設定項目コンポーネント
`components/settings/SettingsItem.tsx`:

```tsx
interface SettingsItemProps {
  label: string;
  value?: string;
  onPress?: () => void;
  rightElement?: React.ReactNode;
}
```

### 4. 設定セクションコンポーネント
`components/settings/SettingsSection.tsx`:

```tsx
interface SettingsSectionProps {
  title: string;
  children: React.ReactNode;
}
```

### 5. 各設定機能の実装

**歩数目標設定:**
- 数値入力（ステッパー）
- プリセットボタン

**テーマ設定:**
- ラジオボタン（ライト/ダーク/システム）
- リアルタイムプレビュー

**重量単位:**
- トグル（kg / lb）

**データエクスポート:**
- JSONエクスポート機能
- Share API使用

**データ削除:**
- 確認ダイアログ
- 全データ削除

**ヘルス連携:**
- 接続状態表示
- 連携解除ボタン

### 6. 永続化
設定値は `expo-sqlite/kv-store` または `AsyncStorage` に保存。
