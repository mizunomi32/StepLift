# Issue #007: 歩数トラッキング画面

## 概要
歩数の表示画面を実装する。目標達成プログレス、統計情報、週間グラフを含む。

## タスク

### 1. 歩数メイン画面
`app/(tabs)/steps.tsx`:

- 円形プログレスリング（目標達成率）
- 今日の歩数表示
- 統計カード（距離、カロリー）
- 週間グラフ
- 目標設定へのリンク

### 2. 円形プログレスコンポーネント
`components/steps/StepGoalRing.tsx`:

```tsx
interface StepGoalRingProps {
  currentSteps: number;
  goalSteps: number;
  size?: number;
}
```

機能:
- SVGで円形プログレス描画
- アニメーション（react-native-reanimated）
- 達成時の視覚的フィードバック

### 3. 歩数統計カード
`components/steps/StepStats.tsx`:

```tsx
interface StepStatsProps {
  distanceKm: number | null;
  calories: number | null;
}
```

表示:
- 📏 距離: X.X km
- 🔥 カロリー: XXX kcal

### 4. 週間歩数グラフ
`components/steps/WeeklyStepChart.tsx`:

```tsx
interface WeeklyStepChartProps {
  records: StepRecord[];
  goal: number;
}
```

機能:
- 棒グラフで過去7日間の歩数表示
- 目標ラインを表示
- 曜日ラベル（月〜日）

### 5. 目標設定モーダル
`app/steps/goal-setting.tsx`:

- 現在の目標表示
- 数値入力（ステッパーまたはスライダー）
- プリセット（5,000 / 8,000 / 10,000 / 15,000）
- 保存ボタン
