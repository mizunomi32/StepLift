# StepLift - 製品要件定義書 (PRD)

## 概要

**StepLift** は、筋トレのワークアウト記録と歩数計を統合したフィットネスアプリです。初心者から上級者まで幅広いユーザーに対応し、日々のトレーニングと活動量を一元管理できます。

## ビジョン

「すべての一歩とすべての一回を、成長の記録に」

## ターゲットユーザー

### プライマリ
- **筋トレ初心者**: シンプルな記録機能で習慣化をサポート
- **中級〜上級者**: 詳細な記録・分析機能でトレーニング最適化

### ユースケース
1. ジムでのワークアウト中に種目・重量・回数を素早く記録
2. 日常の歩数を自動トラッキングし、活動量を可視化
3. 週次・月次で進捗を振り返り、モチベーション維持

## プラットフォーム

- iOS (iPhone)
- Android

## MVP機能一覧

### 1. ワークアウト記録 (優先度: 高)

| 機能 | 説明 |
|------|------|
| 種目選択 | プリセット種目から選択、カスタム種目追加可能 |
| セット記録 | 重量 (kg)、回数 (reps)、セット数 |
| メモ | 各ワークアウトにメモを追加 |
| 履歴表示 | 過去のワークアウト履歴を日付別に表示 |
| テンプレート | よく使うワークアウトメニューを保存・呼び出し |

#### 種目カテゴリ
- 胸 (Chest)
- 背中 (Back)
- 肩 (Shoulders)
- 腕 (Arms)
- 脚 (Legs)
- 体幹 (Core)
- 有酸素 (Cardio)

### 2. 歩数トラッキング (優先度: 高)

| 機能 | 説明 |
|------|------|
| 歩数計測 | デバイスセンサーまたはHealth API連携で自動計測 |
| 日次目標 | デフォルト10,000歩、カスタマイズ可能 |
| 履歴表示 | 日別・週別・月別の歩数履歴 |
| 距離換算 | 歩数から推定距離 (km) を算出 |
| カロリー換算 | 歩数から消費カロリーを推定 |

#### Health API連携
- iOS: HealthKit
- Android: Health Connect

### 3. 進捗ダッシュボード (優先度: 高)

| 機能 | 説明 |
|------|------|
| 今日のサマリー | 本日の歩数・ワークアウト概要 |
| 週間グラフ | 過去7日間の活動推移 |
| 月間カレンダー | トレーニング実施日をカレンダー表示 |
| ストリーク | 連続トレーニング日数 |
| 種目別PR | 各種目の自己ベスト (Personal Record) 表示 |

## デザイン方針

### テーマ
- **ダークモード基調**: ジム環境での使いやすさ、バッテリー消費軽減
- ライトモードも対応（システム設定に追従）

### カラーパレット

| 用途 | ダークモード | ライトモード |
|------|--------------|--------------|
| 背景 | #0A0A0A | #FFFFFF |
| カード背景 | #1C1C1E | #F2F2F7 |
| プライマリ | #22C55E (緑) | #16A34A |
| セカンダリ | #3B82F6 (青) | #2563EB |
| アクセント | #F59E0B (オレンジ) | #D97706 |
| テキスト | #FFFFFF | #000000 |
| サブテキスト | #9CA3AF | #6B7280 |

### デザイン原則
1. **速さ**: ワークアウト中に素早く記録できる
2. **明瞭さ**: 一目で情報が把握できる
3. **モチベーション**: 進捗が可視化され達成感を得られる

## データ保存

### Phase 1 (MVP)
- ローカルストレージ (SQLite via expo-sqlite)
- デバイス内でのみデータ保持

### Phase 2
- クラウド同期オプション追加
- 複数デバイス間でのデータ同期
- バックアップ・リストア機能
